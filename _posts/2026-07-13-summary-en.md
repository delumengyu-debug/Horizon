The model used 64 sub-agents working in parallel and produced a 3-page PDF proof. This marks the first time an AI has autonomously solved a long-standing open problem in mathematics, demonstrating significant progress in automated reasoning and large-scale agent coordination. It could transform how mathematical research is conducted, potentially accelerating discoveries and changing the role of human mathematicians. The conjecture states that every bridgeless graph has a collection of cycles covering each edge exactly twice. GPT-5.6 transformed the problem into edge labeling and linear equations over finite fields, and its prompt required dynamic sub-agent allocation and independent verification without fixed solution steps.

## [Cloudflare Fixes Race Condition in hyper HTTP/1 Library](https://www.infoq.com/news/2026/07/cloudflare-hyper-bug-fix/?utm_campaign=infoq_content&utm_source=infoq&utm_medium=feed&utm_term=global) ⭐️ 8.0/10

Cloudflare discovered and fixed a rare race condition in the hyper Rust HTTP library's HTTP/1 implementation that could silently truncate large responses while returning a 200 OK status. The bug had existed for years and was triggered only under specific timing conditions. This is significant because hyper is a foundational library used by many Rust web frameworks and services. The bug could cause data corruption in large HTTP responses without any error indication, affecting reliability and data integrity. The race condition occurred in hyper's HTTP/1 parsing code when handling chunked transfer encoding or large bodies under concurrent I/O. Cloudflare's team reported and contributed the fix upstream to the hyper repository.

rss · InfoQ · Jul 12, 06:18

**Background**: hyper is a low-level HTTP library written in Rust, known for its performance and safety guarantees. It serves as a building block for higher-level frameworks like axum and warp. Race conditions occur when multiple threads operate on shared data without proper synchronization, leading to unpredictable behavior such as data corruption.

Cross-family fact-checking reduced policy-violating approvals by 45% compared to only 6% when one model played both roles.
